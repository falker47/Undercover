# Undercover

A social deduction party game for 3-12 players on a single device. No internet required.

## How it works

Each player receives a secret word. **Mr. White** has no word and must bluff. **Infiltrators** receive a different word but don't know they're impostors. Players give one-word clues to describe their word, then vote to eliminate the suspect. Find the impostors before they outnumber you!

## Play

**[Play now](https://falker47.github.io/undercover/)**

Install it as a PWA on your phone for the best experience.

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion
- Zustand (state management)
- Vite 5 + vite-plugin-pwa
